Ajuda do ISO Constructor

ISO Constructor helps building your custom Debian ISO, and was originally created for maintaining SolydXK ISOs, but it might benefit those who want to build their custom Debian ISO.

1. Adicionar distribuição

Unpack an ISO

Procure pelo ISO descarregado e seleccione um directório para desempacotar o ISO.

O botão para procurar o directório do desempacotado será disponível se o caminho para o ISO existir.

Add an existing directory

Se você previamente desempacotou um ISO e removeu a distribuição da lista do ISO Constructor, você pode adicionar o directório existente desmarcando a caixa de selecção "Criar do ISO"

O botão Salvar está disponível se o caminho para o ISO e o directório dos descompactados existirem.

2. Remover distribuições

Isso só removerá as distribuições seleccionadas. O directório de distribuição e seu conteúdo NÃO serão excluídos.

3. Abrir directório

Abre o directório de trabalho seleccionado no gestor de ficheiros padrão.

4. Editar no terminal

Abra um terminal chroot e comece a instalar (e remover) todos os pacotes que você precisa. É mais fácil mudar a configuração no directório raiz da distribuição e não através desse terminal.

5. Atualizar as distribuições

Simplesmente execute "apt-get dist-upgrade", mas tendo em conta que alguns serviços precisam ser manipulados antes e depois da atualização.

6. Construir ISOs

Isto consiste nas seguintes etapas:

  • Fazer o download de pacotes off-line (/lib/live/mount/medium/offline).
  • Instalar todos os drivers de firmware.
  • Remover os pacotes órfãos e indisponíveis (incluindo o kernel antigo).
  • Configuração do sistema.

O processo de construção pode levar muito tempo. Se precisar de encerrar o processo antes que ele termine, tem de abrir um terminal e executar:

sudo killall mksquashfs

Você precisa repetir isto se tiver mais do que uma distribuição seleccionada na lista.

7. Localização dos ISOs

Isto irá localizar o seu ISO, definir idioma e fuso horário e consiste em 6 etapas diferentes:

  1. Definir a região desejada.

  2. Definir a localidade padrão.

  3. Definir o fuso horário.

  4. Alterar a configuração Live (/etc/live/config.conf).
  5. Actualizar (apt-get dist-upgrade).
  6. Quando necessário, instale pacotes para o KDE, Firefox, Thunderbird, LibreOffice e outros pacotes de localização. Verifique o script setlocale.sh para pacotes específicos de localidade.

Não esquecer de traduzir o translate boot/boot/grub/grub.cfg and boot/isolinux/isolinux.cfg do ISO.

8. Usar como base o Debian live

Você pode até usar um Debian Live DVD como base: http: //www.debian.org/CD/live Se você usa o Debian, você precisará do diretório "isolinux" (e o seu conteúdo) para substituir o "isolinux" No directório de descompactados Debian. Depois disso, você precisará de configurar a distribuição a partir do zero. É muito trabalho, mas para um CE (Community Edition) vale a pena.

9. Ficheiros importantes

  • ~/.constructor/distributions.list: aqui estão todos os directórios de distribuição salvos.
  • /usr/share/iso-constructor/grubgen.sh: gera boot/boot/grub/grub.cfg (com o grub-template).
  • /usr/share/iso-constructor/isolinuxgen.sh: gera boot/isolinux/isolinux.cfg (com o isolinux-template).
  • /usr/share/iso-constructor/cleanup.sh: este script primeiro limpa e depois cria o ISO.
  • /usr/share/iso-constructor/setlocale.sh: Define o local do ISO. Por defeito é: en_US
  • /usr/share/iso-constructor/excludes: lista com diretórios excluídos da compilação ISO.
