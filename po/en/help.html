ISO Constructor Help

ISO Constructor is a tool to build and maintain custom Debian Live ISOs.

1. Add distribution

Unpack an ISO

Browse for the downloaded ISO, and select a directory to unpack the ISO to.

The button to browse for the unpack directory comes available if the path to the ISO exists.

Add an existing directory

If you previously unpacked an ISO, and removed the distribution from ISO Constructor's list, you can add the existing directory by unchecking the "Create from ISO" checkbox.

The Save button comes available if both the path to the ISO, and the unpack directory exist.

2. Remove distributions

This will only remove the selected distributions. The distribution directory, and its contents will NOT be deleted.

3. Edit distributions

The embedded terminal will start a chroot session where you can run any commands you wish to customize your distribution.

To exit the chroot session, hit Ctrl-D (enter), or type: exit (enter).

4. Upgrade distributions

Simply runs "apt-get dist-upgrade" but taking into account that some services need to be handled before and after the upgrade.

5. Localize the ISOs

ISO Constructor by default generates a boot menu where the user can select their own locale when starting a live session.

When you wish to localize the distribution to a specific locale you can use this option.

  1. Set the desired locale.

  2. Set the default locale.

  3. Set the time zone.

The following actions are automated:

  • Change the live configuration if available.
  • When needed, install packages for KDE, Firefox, Thunderbird, LibreOffice and other localization packages.

Do not forget to translate the Grub and Isolinux menu. See the note below on editing the Grub and Isolinux templates.

6. Build ISOs

This consists of these steps:

  • Download offline packages if needed.
  • Install firmware drivers.
  • Cleanup the system: remove orphaned and unavailable packages.
  • Build a bootable system from root.
  • Configure Grub and Isolinux.*
  • Build the ISO and generate a sha256 file.

* To customize the Grub and Isolinux boot menus you can create and edit the templates in ~/.iso-constructor/:

cp /usr/share/iso-constructor/grub-template ~/.iso-constructor/

cp /usr/share/iso-constructor/isolinux-template ~/.iso-constructor/

7. Important files

  • ~/.constructor/distributions.list: list with distribution directories.
  • ~/.constructor/iso-constructor.log: log file.
  • ~/.constructor/grub-template (optional): custom Grub template.
  • ~/.constructor/isolinux-template (optional): custom Isolinux template.
